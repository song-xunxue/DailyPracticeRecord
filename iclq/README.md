# iclq — 2025年小学期程序设计实践

2025年北京理工大学小学期程序设计实践课程题目解答，共约30道算法题，按周分类。

## 周次与主题

- **第一周**：基础练习（猜数、北湖挖坑、A+B、发言统计等）
- **第二周**：贪心/排序、简单数据结构、查找与搜索
- **第三周**：动态规划
